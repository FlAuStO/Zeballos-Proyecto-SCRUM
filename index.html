Gestión de Calidad de Software

Fundamentos

La calidad de software busca asegurar que un producto cumpla estándares y expectativas.

Técnicas de Aseguramiento

  • Pruebas unitarias y de integración
  • Revisiones de código
  • Automatización de pruebas

Herramientas

Ejemplos: JUnit, Selenium, SonarQube.
